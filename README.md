# 📸 Instagram con Django

Este es un proyecto desarrollado con **Django** que simula las principales funcionalidades de Instagram, incluyendo gestión de perfiles, publicaciones y plantillas HTML. 

## 🚀 Funcionalidades

- Gestión de usuarios y perfiles.
- Creación, edición y eliminación de publicaciones.
- Sistema de plantillas para una experiencia visual.
- Navegación mediante rutas configuradas en las `urls`.
