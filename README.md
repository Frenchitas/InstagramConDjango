# 📸 Instagram con Django

Este es un proyecto desarrollado con **Django** que simula las principales funcionalidades de Instagram, incluyendo gestión de perfiles, publicaciones y plantillas HTML. 

## 🚀 Funcionalidades

- Gestión de usuarios y perfiles.
- Creación, edición y eliminación de publicaciones.
- Sistema de plantillas para una experiencia visual.
- Navegación mediante rutas configuradas en las `urls`.

## 🛠 Tecnologías utilizadas

El proyecto está desarrollado utilizando las siguientes tecnologías:

- **Python 3.8+**: Lenguaje de programación base.
- **Django 4.x**: Framework para el desarrollo web.
- **SQLite**: Base de datos ligera para el desarrollo y pruebas.
- **Bootstrap**: (si aplica) Framework CSS para el diseño responsivo.
- **JavaScript**: Funcionalidades dinámicas (pendiente de detalles).
- **Git y GitHub**: Control de versiones y repositorio remoto.
